# SGO-Projetos-de-Software
Trabalho da disciplina de Projetos de Software - 4° Periodo.

## Sobre o sistema
O Sistema de Gestao de Olimpiadas (SGO) organiza e acompanha competicoes olimpicas academicas. Ele centraliza o cadastro de participantes, equipes, provas e resultados, oferecendo uma visao unificada para planejamento e acompanhamento do evento.
