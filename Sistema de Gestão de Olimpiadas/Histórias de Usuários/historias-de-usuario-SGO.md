## 👥 Usuário

| ID | História de Usuário |
|----|---------------------|
| **US01** | Como **usuário**, quero **fazer login no sistema informando meu e-mail e senha**, para **acessar as funcionalidades disponíveis ao meu perfil**. |
| **US06** | Como **usuário autenticado**, quero **consultar as competições disponíveis com data, horário e local**, para **me manter informado sobre as provas das Olimpíadas**. |
| **US09** | Como **usuário autenticado**, quero **visualizar todas as inscrições realizadas nas competições**, para **acompanhar a situação das participações no sistema**. |

<a id="atleta"></a>
## 🏅 Atleta

| ID | História de Usuário |
|----|---------------------|
| **US02** | Como **atleta**, quero **me inscrever em competições específicas representando meu país**, para **que eu possa participar das provas das Olimpíadas**. |
| **US08** | Como **atleta**, quero **me cadastrar no sistema informando meus dados pessoais e meu país de origem**, para **que eu possa me inscrever nas competições**. |
| **US10** | Como **atleta**, quero **cancelar minha inscrição em uma competição**, para **que minha vaga fique disponível para outro atleta**. |
| **US15** | Como **atleta**, quero **visualizar o histórico de todas as competições em que participei com meus respectivos resultados**, para **acompanhar meu desempenho nas Olimpíadas**. |

<a id="organizador"></a>
## 🗂️ Organizador

| ID | História de Usuário |
|----|---------------------|
| **US03** | Como **organizador**, quero **alocar locais para as competições verificando conflitos de horário**, para **garantir que um local não seja usado por duas competições ao mesmo tempo**. |
| **US04** | Como **organizador**, quero **registrar os resultados das competições indicando o primeiro, segundo e terceiro colocados**, para **manter o controle oficial dos resultados**. |
| **US05** | Como **organizador**, quero **gerar relatórios de medalhas por país mostrando ouro, prata e bronze conquistados**, para **acompanhar o desempenho de cada nação nas Olimpíadas**. |
| **US07** | Como **organizador**, quero **cadastrar competições informando o nome da modalidade, data, horário, local e lista de atletas inscritos**, para **que as competições fiquem disponíveis no sistema**. |
| **US11** | Como **organizador**, quero **visualizar um painel com todos os resultados registrados por modalidade**, para **ter uma visão geral do andamento das Olimpíadas**. |
| **US12** | Como **organizador**, quero **cadastrar locais informando nome, endereço e capacidade**, para **que fiquem disponíveis para alocação nas competições**. |
| **US13** | Como **organizador**, quero **consultar a disponibilidade de um local em uma data e horário específicos**, para **evitar conflitos na alocação**. |
| **US16** | Como **organizador**, quero **editar os dados de uma competição já cadastrada**, para **corrigir informações caso necessário**. |
| **US17** | Como **organizador**, quero **cancelar uma competição cadastrada**, para **remover provas que não serão realizadas e notificar os atletas inscritos**. |
